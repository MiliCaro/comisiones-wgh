# Comisiones WGH Argentina (portal)

Landing-portal que reúne las **4 comisiones** de Women in Global Health Argentina y el **reglamento de funcionamiento** transversal a todas.

Página HTML self-contained (un solo archivo, sin paso de build). Lista para publicar en GitHub Pages.

## Identidad
- Acento del portal: **aqua `#96D6DF`** (texto midnight en botones)
- Cada comisión conserva su color propio en su tarjeta: Consulta `#E8512A`, Coloquio `#C8E535`, Comunidad `#C2185B`, Alianza `#29B6D4`
- Tipografía: Inter

## Contenido
- Grilla con las 4 comisiones, enlazando a cada landing publicada.
- **Reglamento de las comisiones** (transversal): criterios de participación, equipo operativo (~8), 75% de asistencia, roles distribuidos, rol del liderazgo y nota de construcción colectiva.

## Publicar en GitHub Pages
1. Subí este repo a GitHub (usuario `milicaro`).
2. Settings → Pages → Source: `main` / `/ (root)`.
3. La página queda en `https://milicaro.github.io/comisiones-wgh/`.
